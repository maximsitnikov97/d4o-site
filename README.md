# D4O — Сайт продуктового IT-агентства

Конверсионный лендинг для [d4o.tech](https://d4o.tech).

## Стек

Next.js 16 · React 19 · TypeScript · Tailwind CSS 4 · Framer Motion · shadcn/ui

## Быстрый старт

```bash
npm install
npm run dev
```

Открыть [http://localhost:3000](http://localhost:3000)

## Команды

| Команда | Описание |
|---------|----------|
| `npm run dev` | Dev-сервер |
| `npm run build` | Production-билд |
| `npm run start` | Запуск production |
| `npm run lint` | ESLint |

## Структура

```
src/
├── app/              # Next.js App Router
├── components/
│   ├── sections/     # Секции лендинга (Header, Hero, Services...)
│   └── ui/           # shadcn компоненты
└── lib/              # Утилиты
```
